## Exposure metric

Flood exposure at point-based power assets was estimated by extracting flood depth statistics within a buffer surrounding each asset location. A buffer radius of XX m was used to account for asset footprint and spatial uncertainty. The 95th percentile flood depth was selected as the representative exposure metric to ensure robustness against single-pixel artefacts while capturing severe local flooding.


## Flood depth raster

There is no spatially continuous observed depth raster in practice.
Flood depth rasters at scale are based on simulations, which can be adjusted from observations.
Hydraulic models are physics-based which enforces realism.
